## 6대 핵심 업데이트 (영상 메인 토픽)

### 1. Voice Mode — 말로 하는 코딩

- 스페이스바 Push-to-Talk, 20개 언어
- **커뮤니티**: "타이핑보다 빠르다" (Medium), RSI 환자 접근성 혁신
- **기획자 관점**: 터미널 진입장벽을 음성이 낮춤. "브리프를 말로 주는 것"

### 2. Claude Code Channels — 텔레그램/디스코드 연동

- 3/20 Research Preview. 메신저에서 실행 중 세션에 메시지 전송
- **커뮤니티**: "OpenClaw 킬러" (VentureBeat), 이동 중 코딩 세션 제어
- **기획자 관점**: 외출 중에도 휴대폰으로 AI에게 작업 지시. "외주한테 카톡 보내듯"

### 3. 100만 토큰 컨텍스트 + Opus 4.6

- 3/13. 기존 200K의 5배. Max/Team/Enterprise 기본 활성화
- 출력 한도: 64K 기본, 128K 상한
- **기획자 관점**: 전체 프로젝트를 한 번에 이해시킬 수 있음. 맥락 끊김 없음

### 4. /loop — 반복 실행 스케줄러

- 3/7. `/loop 5m check the deploy` 형태. 세션 내 경량 크론
- 최대 50개 동시, 3일 후 자동 만료
- **기획자 관점**: "5분마다 배포 상태 확인해줘"를 자연어로 설정

### 5. /effort — 사고 깊이 조절

- 3/14. Low(빠른) / Medium(기본) / High(심층) 3단계
- "ultrathink" 키워드로 일시적 High 활성화
- **기획자 관점**: 간단한 질문 vs 복잡한 분석을 직접 조절

### 6. MCP Elicitation — AI가 폼으로 물어봄

- 3/14. MCP 서버가 작업 중 구조화된 입력 요청
- **기획자 관점**: AI가 작업 중 "이거 어떻게 할까요?" 폼으로 물어보는 자연스러운 대화형 워크플로우
