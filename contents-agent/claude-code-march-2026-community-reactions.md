## 1. 3월 주요 업데이트 요약

| Voice Mode (`/voice`) | 푸시투톡 방식 음성 입력. 스페이스바를 누르고 말하면 텍스트로 변환되어 전송 |

## 3. 사람들이 가장 열광한 기능

| 순위 | 기능 | 왜 열광했는가 |
|------|------|-------------|
| 1 | **Voice Mode** | 타이핑 없이 대화로 코딩. 접근성 혁신. "타이핑보다 빠르다"는 반응 |
| 2 | **Claude Code Channels** | Telegram/Discord에서 이동 중 코딩 세션 제어. OpenClaw 대안 가능성 |
| 3 | **100만 토큰 컨텍스트** | 대규모 코드베이스 전체를 한 번에 이해 가능 |
| 4 | **Code Review** | PR당 $0.10 미만으로 200개+ 리뷰 가능. 비용 효율 극대화 |
| 5 | **사용량 2배 프로모션** | 특히 한국에서 체감 효과가 매우 큼 |

## 4. 주요 불만 및 불편 사항

| 불만 | 상세 |
|------|------|
| **사용량 한도** | 가장 큰 불만. "리듬 타는 순간 끊긴다." 5시간 롤링 윈도우 + 주간 캡 + 분당 RPM 한도가 서로 소통하지 않음 |
| **가격 구조** | $20 Pro 플랜이 하루 2~3회 중단되면서 오히려 생산성 손실 비용이 더 큼 |
| **Channels 한계** | "OpenClaw를 완전히 대체하기엔 아직 벽 몇 개가 부족하다" |
| **바이브 코딩 부작용** | 비개발자가 만든 저품질 AI 코드가 오픈소스에 범람. Gentoo Linux, NetBSD는 AI 생성 기여물 금지 |
| **생산성 패닉** | Bloomberg 보도: AI 코딩이 쉬워진 게 아니라 "어떤 대가든 빌드해야 하는" 고압적 경쟁 촉발 |

## 5. 경쟁 도구 비교 (커뮤니티 컨센서스)

| 도구 | 강점 | 약점 | 사용 시나리오 |
|------|------|------|-------------|
| **Claude Code** | 코드 품질 최고 (67% 승률), 아키텍처 이해력 | 사용량 한도, 비쌈 | 복잡한 작업, 전체 코드베이스 이해 필요 시 |
| **Cursor** | IDE 네이티브, 기존 코드 스타일 학습 | Claude Code 대비 코드 품질 하락 | 일상적 코딩, 빠른 인라인 제안 |
| **GitHub Copilot** | 플러그인 방식 접근 편의성 | 자동완성이 더 이상 최고가 아님 | 기존 에디터 위 가벼운 보조 |
| **Codex (OpenAI)** | 사용량 한도 여유 | Claude 대비 품질 낮음 | 장시간 연속 작업 필요 시 |

> "가장 생산적인 개발자들은 이미 두 도구를 병행한다" — Dev.to 조사

## 6. 비개발자/기획자/디자이너 관련 반응 (영상 타겟 핵심)

### 진입 장벽이 낮아졌다
- 한국에서 "비개발자가 클로드코드로 코딩하는 법" 35분 입문 가이드가 존재
- 비개발자 마케터가 크리에이티브 아웃풋을 **10배** 늘린 사례 보고
- Voice Mode로 터미널 입력 자체가 "대화"로 전환 — 코딩 경험 없어도 접근 가능

### 자동화 실용 사례
- **리포팅 자동화**: 여러 소스에서 데이터 수집 → 일관된 형식 변환 → 내러티브 작성 → 클라이언트 패키징 (에이전시 최대 고통점)
- **콘텐츠 운영**: 유튜브 영상을 멀티 플랫폼 소셜 포스트로 자동 변환
- **사용자 리서치**: 인터뷰 트랜스크립트 자동 분석 (UX Planet 기사)
- **내부 도구 제작**: 비개발자도 업무용 도구를 직접 빌드

### "바이브 코딩" 성숙
- 단순 유행을 넘어 실무 도구로 안착 중
- 다만 품질 관리 없이 남용하면 오픈소스 생태계에 부정적 영향

### 디자이너 특화
- [UX Planet — Claude Skills for Product Designers](https://uxplanet.org/claude-skills-for-product-designers-a453a7a8faa7): 디자인 시스템 철학을 코드에 반영, 의도적인 타이포그래피/색상/애니메이션 출력
- Anthropic 공식 `frontend-design` 스킬이 디자인 시스템과 철학을 코드 전에 주입
