## 활용 파이프라인

```
1. 소스 영상 transcript 추출 (firecrawl scrape → YouTube transcript)
2. 핵심 정보 추출 (팩트, 수치, 인사이트, 비유)
3. 메이커 에반 스타일 + 커스텀 캐릭터로 재구성
4. 한국 시청자 맥락에 맞게 로컬라이징
5. 씬 분할 대본 생성
```
