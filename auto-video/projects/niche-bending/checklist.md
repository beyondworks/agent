# AI 시네마틱 다큐 — 30가지 제작 체크리스트

> 성공 채널(와일드/TierZoo/ColdFusion)의 성공 요인 + 실패 채널(AI 슬롭/딸깍 자동화)의 실패 요인 분석 기반
> 통과 기준: 120점/150점 이상

---

## 퀄리티 (1~5)

| # | 체크 항목 | 근거 | 0~5 |
|---|----------|------|-----|
| 1 | 나레이션이 사람 목소리처럼 자연스러운가 (감정/리듬 변화) | 단조 TTS는 첫 45초 이탈률 35% 증가 | |
| 2 | 영상 해상도 1080p 이상, 모바일에서 선명한가 | 저화질 = 저품질 신호 → 즉시 이탈 | |
| 3 | 오디오 믹싱: 나레이션/배경음악/효과음 균형 | 음악이 나레이션 덮으면 즉각 이탈 | |
| 4 | AI 생성 이미지/영상에 아티팩트(결함) 없는가 | 검수 없는 자동화 채널 = 2026.01 단속 대상 | |
| 5 | 자막 싱크/가독성 확보 (모바일 최적화) | 싱크 오류 = AI 슬롭 신호 | |

## 스토리텔링 (6~10)

| # | 체크 항목 | 근거 | 0~5 |
|---|----------|------|-----|
| 6 | 첫 30초 안에 후크(질문/충격/갈등)가 있는가 | 시청자 55%가 60초 내 이탈 | |
| 7 | 정해진 서사 구조(기승전결)를 따르는가 | 위키 요약 나열 → 이탈률 폭등 (실패 패턴) | |
| 8 | 감정적 고점/저점이 설계됐는가 | 단조로운 톤 = AI 슬롭 대표 특징 | |
| 9 | 독창적 관점/포맷이 있는가 | TierZoo: 게임 프레임 = 388만 구독 핵심 이유 | |
| 10 | 영상 길이가 내용 밀도에 최적화됐는가 (패딩 없음) | 광고용 인위적 늘리기 → 후반 이탈 급증 | |

## 팩트/신뢰 (11~15)

| # | 체크 항목 | 근거 | 0~5 |
|---|----------|------|-----|
| 11 | 모든 사실(숫자/날짜/인명)이 검증됐는가 | AI 환각 → 댓글 지적 → 신뢰 영구 손상 | |
| 12 | 최신 정보를 사용하고 있는가 (1년 이상 된 데이터 주의) | AI 훈련 데이터 컷오프 → 오래된 정보를 최신처럼 제시 | |
| 13 | AI 생성 콘텐츠임을 유튜브 정책에 따라 공개했는가 | 미공개 시 수익화 정지/채널 종료 위험 (2025.03 정책) | |
| 14 | 논란 주제에서 균형 잡힌 시각인가 | 편향 과장 → 커뮤니티 가이드라인 위반 신고 | |
| 15 | 시청자 팩트 지적에 대응 체계가 있는가 | 오류 방치 → 알고리즘이 "낮은 만족도"로 판단 | |

## 기획/소재 (16~20)

| # | 체크 항목 | 근거 | 0~5 |
|---|----------|------|-----|
| 16 | 명확한 마이크로 니치를 갖고 있는가 | 넓은 주제 = 사망 선고. 니치 1위 = 추천 신뢰 신호 | |
| 17 | 검색 수요 + 추천 피드 클릭 가능성을 동시에 고려했는가 | 둘 중 하나만 = 절반의 성장 | |
| 18 | 시리즈/연속물 구조로 재방문율을 설계했는가 | 채널 내 연속 시청 = 강력한 알고리즘 신뢰 신호 | |
| 19 | 트렌드(빠른 뷰) + 에버그린(지속 유입) 비율이 균형인가 | 트렌드만 = 알고리즘 변화에 취약 | |
| 20 | 동일 포맷 대량 반복을 하지 않는가 | 2026.01 단속: 16개 채널(3500만 구독) 하루에 삭제 | |

| 23 | AI 생성 이미지/영상의 시각 스타일이 일관되는가 | 무작위 스타일 = "AI 스팸" 인식 | |
| 24 | 편집 리듬(컷/전환)이 콘텐츠에 맞게 설계됐는가 | 기계적 고정 간격 컷 = 지루함 → 이탈 | |
| 25 | 썸네일 A/B 테스트를 활용하는가 | A/B 테스트 → 평균 CTR 20% 향상 | |
