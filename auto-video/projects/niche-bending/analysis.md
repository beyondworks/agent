---

## 핵심 공식

```
유튜브 = 시장(Market) + 포맷(Format)

니치 밴딩 = 시장A에서 검증된 포맷을 시장B에 적용
         = 블루오션을 찾는 게 아니라 직접 만드는 것
```

- **시장**: 금융, 운동, 게임, 역사, 뷰티, 인테리어 등 (거의 변하지 않음)
- **포맷**: 콘텐츠를 담는 그릇 (매일 새롭게 탄생, 천재들이 만듦)

---

## 검증된 포맷 4가지

### 1. POV (시점 영상)
- "당신이 ~라면" / "POV: ~"
- 사례: 게임 NPC 시점 → 269만뷰 (구독자 3,000명 채널)
- 마인크래프트 POV → 377만, 286만, 234만뷰
- 스파이더맨 POV → 27만뷰 (구독자 5,000명)
- 특징: 몰입감 극대화, 숏폼에 특히 강함

### 2. 2D 애니메이션 + "당신의 삶이 ~라면"
- 두 가지 포맷의 결합
- 사례: "뉴얼라이프가 ~라면" + 2D 애니메이션 → 515만뷰
- "보스의 아들" 주제 변경 → 3주 만에 63만뷰
- 특징: AI 도구로 제작 가능, 높은 공감 유발

### 3. 3D 애니메이션
- AI 영상 도구(Grok, Hailo, BearSri, Midjourney)로 제작 가능
- 사례: 팀 다닐로브 — 유튜버 작동구조를 병맛 3D로 표현 → 구독자 87만
- 원본: 잭 필름 (구독자 2,500만) 포맷을 가져옴
- 피트니스 시장에 적용 → 엄청난 성과
- 특징: 1년 전까지 전문가 영역이었으나 AI로 진입장벽 소멸

### 4. 게임 해설 포맷
- 공격력/방어력/티어리스트 등 게임 용어로 비게임 주제 설명
- 사례: 티어주(TierZoo) — 동물 다큐를 게임 해설로 → 전 영상 100만뷰+
- 내셔널지오그래픽 같은 정통 다큐 대신 게임 포맷으로 → 독보적 포지션
- 특징: 딱딱한 시장을 재미있게 만드는 데 최적

## 영상에서 제안한 니치 밴딩 조합 4가지

### 1. 인테리어 × 2D 유인원/바나나
- 원숭이+바나나 포맷은 금융/피트니스에서 수백만뷰 검증
- 인테리어 시장에 미적용
- 조명 배치, 좁은 방 넓게 보이는 법 등을 유쾌한 원숭이로 설명

### 2. 역사 × 게임 해설
- 역사 카테고리에 게임 설명/티어리스트 포맷
- 로마 제국 멸망 = 게임 개발자의 하향 업데이트
- 고대 국가 군사력 티어리스트

### 3. 직장생활 × POV
- "당신이 IT 스타트업 말단 직원이라면" 같은 꽁트
- 수백만뷰 검증된 POV 포맷 + 딱딱한 직장 시장
- 독보적 재미

### 4. 비즈니스 × 2D 애니메이션 계급 설명
- 범죄 조직 내부 운영 → 2D 애니메이션으로 첫 달 만에 대규모 성과
- 비즈니스 시장에 적용: 글로벌 대기업 숨겨진 계급, 협상 레벨 1~5단계

## 우리 프로젝트에 적용 가능한 조합 (검토 필요)

> 이 프로젝트에서 활용 가능한 자산: AI 영상 생성(Grok), 캐릭터 일러스트(Gemini), TTS(Voicebox), auto-video 파이프라인

### 후보 A: IT/테크 × 게임 해설
- AI 모델별 공격력/방어력/티어리스트 비교
- ChatGPT vs Claude vs Gemini "배틀"
- 프로그래밍 언어 티어리스트
- **장점**: IT 지식 활용 가능, 게임 해설 포맷 검증됨

### 후보 B: 직장생활 × 3D 애니메이션
- IT 직장인 일상을 3D 캐릭터로 표현
- "개발자의 하루", "PM이 무한 요구사항 추가하는 과정"
- Grok 3D 영상 생성 활용 가능

### 후보 C: 심리/자기계발 × POV
- "당신이 번아웃에 빠진 개발자라면"
- "당신의 뇌가 유튜브 알고리즘에 중독됐다면"
- 공감 + 몰입 극대화

### 후보 D: AI/테크 뉴스 × 2D 애니메이션 캐릭터
- AI 뉴스를 2D 캐릭터들이 설명하는 포맷
- 캐릭터 IP 확장 가능 (애덕이와 별도)
