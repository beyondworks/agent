# 유튜버 "와일드" (Wild) 채널 분석

출처: https://youtu.be/oSggY4PIBEY (맥스 채널 인터뷰)

---

## 프로필

- **이름**: 와일드 (Wild)
- **본업**: 회사원 (부업으로 유튜브)
- **배경**: 10년+ 직장생활 → 야근 과다로 중증 장애 → 병원에서 AI+유튜브 접목 결심
- **학습**: 독학 + 무료 강의. AI 툴에 대한 투자에 집중 (최고급 요금제 전부 테스트)

- 7,000만뷰 영상 있음
- 6,000만뷰 영상 있음
- 대부분 1,000만뷰 이상
- 100만뷰 미만 영상은 거의 없음
- 단일 영상 수익: 400~800만원 (광고 + 음원)

## 콘텐츠 유형

### 메인: AI 다큐멘터리 (글로벌 타겟)
- 실제 역사 사건을 AI 영상으로 다큐멘터리화
- 예시: 진주만 습격 → AI 다큐 (원본: 한국 유튜버 580만뷰 썰풀기 → 와일드: AI 다큐로 변환)
- **실사 0%, 전부 AI 생성** — 그러나 시청자가 AI인지 실사인지 구분 못 할 정도의 퀄리티
- 글로벌 타겟 (영어 나레이션 + 미국 특정 타겟도 별도 운영)

### 서브: 감동 스토리 + 쇼츠
- 동물 구조, 감동 실화 등
- 쇼츠 + 롱폼 혼합

### 뮤직비디오
- AI로 뮤직비디오 제작
- AI 음원 제작 → 배포 → 타인이 쇼츠에 사용 → 음원 수익 발생

### 야화 (한국 타겟)
- 한국어 AI 야화 콘텐츠
- 높은 퀄리티 강조

## 음원 수익 모델 (추가 수익원)

- AI로 음원 제작
- 음원 플랫폼에 등록
- 타인이 해당 음원으로 쇼츠 3,200개+ 제작
- 음원 사용 수익이 와일드에게 귀속
- 월 100만원+ 꾸준히 발생 (업로드 안 해도)
- **주의**: AI 음원 그냥 올리면 삭제됨. "특별한 툴을 거쳐서" 올려야 유지됨

---

## 제작 파이프라인

### 정석 방식 (수동)
1. 레퍼런스 채널에서 떡상 소재 발굴 (한국 채널 → 해외 다큐로 변환)
2. ChatGPT로 대본 각색 + 영어 번역
3. 이미지 프롬프트 생성 (다큐 스타일, 사실적, 통일성 유지)
4. **나노 바나나** (Nano Banana = Gemini 이미지 생성)로 이미지 제작
   - 플로우(Floo) 사이트에서 나노바나나 2 무료 사용 가능
   - 지침 입력으로 스타일 제어 (예: "당시 뉴스 영상처럼")
5. 이미지 → 영상 변환
6. ElevenLabs로 영어 나레이션 생성
7. CapCut에서 편집 (영상 + TTS + 자막)

### 자체 프로그램 (자동화)
- 직접 개발한 올인원 프로그램 사용
- 기능:
  - 실제 사건 중 유튜브 떡상할 소재 AI 탐색 + 대본 작성
  - 이미지 프롬프트 자동 생성
  - 썸네일 제목 10개 자동 제안
  - 썸네일 이미지 프롬프트 생성
  - 이미지 일괄 생성 (한 번에 20장)
  - 영상 변환
  - TTS 음성 생성 (톤 선택 가능)
  - 자막 생성
- 대본 → 이미지 → 영상 → TTS → 썸네일 한 번에 처리
- 비용 거의 안 듦
- **수강생에게도 공개** (프로그램 + 105페이지 전자책 무료 배포)

## 핵심 인사이트

### 1. 퀄리티가 전부
> "AI 저품질 영상은 퇴출 선언, 품질 높은 AI는 적극 지원"
> "자동화로 딸깍해서 영상 만들어 수천만원 벌었다는 강의는 돈 날리는 강의"
> "검수하지 못하는 자동화는 절대 하지 마라"

### 2. 기술 > 수익
> "유튜브 수익만 보고 딸깍만 배우면 망한다"
> "기술이 있어야 유튜브 수익도 있다"
> "퀄리티가 담보되고 기본기가 되면 수익은 자연스럽게 따라온다"

### 3. 니치 밴딩 적용
- 한국 유튜버의 "썰풀기" 콘텐츠(580만뷰) → AI 다큐멘터리로 포맷 전환
- 동일 소재, 다른 포맷 + 글로벌 타겟 = 더 큰 시장

### 4. AI는 시작 단계
> "AI 유튜브는 이제 막 시작하는 단계. 더 늦기 전에 시작하라"
> "퀄리티 있는 영상 + 좋은 기획력이면 충분히 승산 있다"

## 우리 프로젝트 시사점

1. **나노 바나나(Gemini)를 이미 사용 중** — 우리 파이프라인과 동일
2. **CapCut 편집** — 동일 도구
3. **ElevenLabs 나레이션** — 우리는 Voicebox 사용 (차별점)
4. **글로벌 타겟** — 영어 나레이션으로 해외 시장 공략 가능
5. **소재 발굴법**: 한국 레퍼런스 채널의 인기 소재 → AI 다큐로 변환
6. **음원 수익 모델**: AI 음원 제작 → 플랫폼 등록 → 패시브 인컴
7. **자동화 but 검수**: 프로그램으로 효율화하되, 반드시 사람이 검수
